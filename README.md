# Pune-fair-tracker

A lightweight web application that helps users compare estimated fares across major ride-booking platforms in Pune. The tool calculates price ranges based on distance, cab type, and time of day, and allows users to compare these with driver-quoted fares to identify overcharging.

Built with a privacy-first approach, all computations run locally in the browser with no data storage or tracking. The app also suggests cheaper public transport alternatives and provides actionable insights (e.g., fair price vs overpriced) to help users make informed travel decisions.
